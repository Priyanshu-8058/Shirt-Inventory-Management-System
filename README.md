# Shirt-Inventory-Management-System
Developed a Shirt Inventory Management System using Core Java and Collections Framework. Implemented CRUD operations, searching, filtering, grouping, and sorting functionalities based on brand, size, colour, and price. Utilized ArrayList, Map, Comparator, Collections API, equals(), and hashCode() methods to efficiently manage inventory data.

## Technologies Used

- Java
- Collections Framework
- ArrayList
- HashMap
- Comparator
- OOP (Object-Oriented Programming)

## Features

- Add Shirt Records
- View All Shirts
- Search Shirts by Brand
- Search Shirts by Brand and Size
- Search Shirts by Price Range
- Search Shirts by Size and Price Range
- Sort Shirts by Size (Ascending & Descending)
- Sort Shirts by Price (Ascending & Descending)
- Sort Shirts by Brand (Ascending & Descending)
- Multi-Level Sorting using Comparator
- Count Shirts by Brand
- Count Shirts by Colour
- Count Shirts by Size
- Group Shirts by Brand using Map
- Find Costliest and Cheapest Shirt
- Remove Shirts by Brand and Size
- Return Immutable List of Shirts
